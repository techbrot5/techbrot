Brand button — use for every action; primary is ink-filled (oxblood hover), ghost is hairline, text is an underlined inline action with arrow-less restraint.

```jsx
<Button href="/contact/">Speak to an expert</Button>
<Button variant="ghost">See pricing</Button>
<Button variant="text">How it works</Button>
```

Props: `variant` (primary | ghost | text), `size` (sm | md | lg), `inverted` (dark surfaces), `full`, `href`. Labels are calm sentence-case imperatives — never exclamation marks, never emoji.
