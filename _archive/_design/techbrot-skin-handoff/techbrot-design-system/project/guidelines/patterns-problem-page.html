QuickBooks · Error 6189

QuickBooks can't open the company file.

What's happening

Error 6189 fires when the file's transaction log is out of sync with the company file — usually after a crash, a forced shutdown, or two machines opening the same file.

Likely cause

The .TLG log file no longer matches the .QBW. QuickBooks refuses to open rather than risk writing inconsistent data — annoying, but it's protecting your books.

Fix it

  1. 1Close QuickBooks on every machine that can reach the file.
  2. 2Run QuickBooks File Doctor from the official Tool Hub.
  3. 3If it persists, restore last night's verified backup and re-enter today.
  4. 4Verify data (File → Utilities → Verify) before anyone else opens the file.

When to stop and call

No verified backup, multi-user damage, or step 2 loops — repair beats re-entry, and it's faster with the file in front of us.
