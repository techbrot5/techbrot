Tinted pill label for status and categories (e.g. "QuickBooks", "Research", "Overdue").

```jsx
<Badge variant="accent">Framework</Badge>
<Badge variant="positive">Reconciled</Badge>
<Badge>QuickBooks</Badge>
```

Variants: `neutral` (default) · `accent` · `brass` · `positive` · `warning` · `critical`. Sentence case, 1–2 words.
