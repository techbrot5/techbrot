Standard section header — eyebrow + regular-weight Fraunces heading + lede. Use at the top of every page section; left-aligned by default.

```jsx
<SectionHeader
  eyebrow="What we do"
  heading="Bookkeeping that holds up to scrutiny."
  lede="Monthly close, cleanup, payroll, and sales-tax compliance — delivered by a certified team."
/>
```

`centered` for symmetric sections. Headings are sentence case, declarative, never bold.
